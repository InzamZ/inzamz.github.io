F. Obtain Two Zeroes

原题CF1260B

真正的签到题 , 看完 A 题题干后有点方向 , 但由于是几何不敢写 . 看榜发现 6min 有人 A 了 . 于是果断 F 题 .

给你两个数 , 可以让一个数减一 , 同时让另一个减二 , 问是否有可能两个数都减到 0 .

先减到相等 , 然后如果是三的倍数就可以 , 否则不行 . 相等是负数也不行 .

参考代码

#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main()
{
    int T = 1, n;
    ll c, s;
    scanf("%d", &T);
    while (T--) {
        scanf("%lld %lld", &c, &s);
        if (c > s)
            swap(c, s);
        ll res = c - (s - c);
        if (res >= 0 && res % 3 == 0)
            printf("YES\n");
        else
            printf("NO\n");
    }
    return 0;
}

D. Heating

原题CF1260A

才发现原题 D 是放在 F 前面的 . 所以决策太重要了 .

装暖气 , 总共需要 sum 个单位 , 至多可以分成 c 组 . 花费是每一组的单位的平方 , 问最小花费 .

显然尽量平均分 , 无法平均分就把剩余的余数平摊到某些组 .

参考代码

#include <bits/stdc++.h>
using namespace std;
using ll = long long;

int main()
{
    int T = 1;
    ll c, sum;
    scanf("%d", &T);
    while (T--) {
        scanf("%lld %lld", &c, &sum);
        ll res = sum / c;
        ll cnt = sum % c;
        ll ans = 0;
        for (int i = 1; i <= c; i++) {
            if (i <= cnt)
                ans += ((res + 1) * (res + 1));
            else
                ans += ((res) * (res));
        }
        printf("%lld\n", ans);
    }
    return 0;
}

G. Infinite Fence

原题CF1260C

涂色 , 每逢 r 的倍数要涂成红色 , 每逢 b 的倍数要涂成蓝色 , 公倍数都可以 , 如果 k 个相同的木板连续就被处决 . 问可否避免处决 .

一开始只是考虑说大的数的间隔能不能容得下 k 个小的数 . k 个小的数事实只需要个位置 . 但是会出现 4 6 2 这个样例不正确 .

其实我们发现4 6 2 不可能成立是因为4不可能刚好出现在6相邻处 , 这和最大公因数有关系 . 而我们发现4 6的涂色和2 3 的涂色情况是一样的 , 而只要两个数互质 , 就满足上面说的关系 . 所以我们只需要把 r b 都除以两者的最大公因数 , 就可以用上面的公式直接判断了 .

参考代码

#include <bits/stdc++.h>
using namespace std;
using ll = long long;

ll gcd(ll x, ll y)
{
    return y == 0 ? x : gcd(y, x % y);
}

int main()
{
    int T = 1;
    ll r, b, k;
    scanf("%d", &T);
    while (T--) {
        scanf("%lld %lld %lld", &r, &b, &k);
        if (r > b)
            swap(r, b);
        ll gcdd = gcd(r, b);
        r /= gcdd; b /= gcdd;
        ll res = b % r;
        if (r == b || r + 1 == b || r * k - r + 1 >= b)
            printf("OBEY\n");
        else
            printf("REBEL\n");
    }
    return 0;
}
